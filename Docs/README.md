# TidyTunes

TidyTunes is a smart music library organizer that helps clean, identify,
and manage large music collections.

It scans your files, fixes missing information, finds duplicates,
checks audio quality, and safely organizes your library while keeping
your original files protected.

## Features

* Music library scanning
* Metadata analysis and cleanup
* AcoustID fingerprint identification
* MusicBrainz enrichment
* Duplicate detection and resolution
* Audio integrity checking
* Library reports
* Safe organization workflows

## Requirements

* Windows
* .NET 10
* FFmpeg
